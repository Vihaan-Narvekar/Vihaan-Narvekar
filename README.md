## What I'm Building

### Kalshi Weather Market Research System  
A weather-market research pipeline for studying prediction markets tied to temperature outcomes.

- Fetches and parses **Kalshi order books**
- Collects **NWS forecasts** and **NOAA observations**
- Builds **no-leakage datasets** using as-of joins
- Engineers market features like midpoint, spread, microprice, and imbalance
- Designed around the question: *What information was actually available at the time?*

### FOMC LLM Classification Research  
Research work using LLMs to classify central bank communication.

- Built and evaluated an **LLM-based NLP pipeline** for FOMC statements
- Compared **open-source and closed-source models**
- Used rubric-guided classification for Phillips Curve and IS Curve arguments
- Focused on preserving qualitative nuance while scaling classification

### Systematic ETF Trading Strategy  
A quantitative research project forecasting ETF returns across SPY, QQQ, and IWM.

- Built regression-based models using OLS, Ridge, Lasso, and Quantile Regression
- Evaluated out-of-sample performance and directional hit rate
- Translated model outputs into market-timing portfolio positions
